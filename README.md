# 📊 FinançasPro - Sistema de Gestão Inteligente

Projeto acadêmico desenvolvido para a disciplina de Oficina de Programação.

🔗 **[Acesse a Aplicação Online Aqui](https://whiniciusbraz.github.io/Projeto-Finan-as/)**

*(Credenciais de teste: Usuário: `admin` | Senha: `1234`)*

## 🚀 Principais Funcionalidades

* **Autenticação Simulada:** Proteção de rotas e bloqueio de acesso sem credenciais.
* **Motor SPA (Single Page Application):** Navegação fluida entre abas sem recarregar a página.
* **Dashboard Dinâmico:** Resumo de saldo, receitas e despesas filtrados por mês de referência.
* **Múltiplos Cartões de Crédito:** Cadastro dinâmico de cartões com cálculo de limite e faturas individuais.
* **Sistema Especialista de Recomendação:** Motor heurístico (simulação de IA) que analisa os hábitos de gastos em tempo real e fornece conselhos financeiros baseados em regras lógicas.
* **Exportação de Dados:** Geração e download nativo de relatórios em formato `.csv` (Excel).
* **Persistência de Dados:** Uso avançado de `localStorage` para manter os registros salvos no navegador.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Estilização avançada com variáveis (Custom Properties), Flexbox, CSS Grid, animações e responsividade total.
* **JavaScript (ES6+):** Lógica de negócio, manipulação de DOM, formatação de dados (Regex) e controle de estado.
* **Phosphor Icons:** Biblioteca de ícones vetoriais profissionais.

## ⚙️ Como executar o projeto localmente

1. Clone este repositório: `git clone https://github.com/WhiniciusBraz/Projeto-Finan-as.git`
2. Abra a pasta do projeto no VS Code.
3. Inicie a aplicação utilizando a extensão **Live Server** no arquivo `index.html`.)**
